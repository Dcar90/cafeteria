# ☕ Aroma & Grano - Landing Page

Una landing page moderna, elegante y totalmente responsiva diseñada para una cafetería de especialidad. Este proyecto se centra en la experiencia del usuario, utilizando una estética cálida y profesional.

## 🚀 Características

- **Diseño Responsivo:** Optimizado para dispositivos móviles, tablets y escritorio.
- **Tipografía Elegante:** Uso de Google Fonts (`Playfair Display` y `Poppins`) para una jerarquía visual clara.
- **Secciones Clave:**
  - Hero section con llamado a la acción (CTA).
  - Menú de productos con tabla de precios.
  - Sección de valores/características del negocio.
  - Testimonios de clientes.
  - Pie de página con información de contacto y horarios.
- **Código Limpio:** HTML5 semántico y CSS3 moderno (Flexbox y Grid).

## 🛠️ Tecnologías Utilizadas

| Tecnología | Uso |
| **HTML5** | Estructura semántica del contenido. |
| **CSS3** | Estilos, diseño responsivo y animaciones.| 
| **Google Fonts** | Tipografía personalizada.| 
| **Unsplash API**| Imágenes de alta resolución para el diseño.|
